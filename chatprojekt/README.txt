# Mini-Chat-Projekt erstellt von Mulla Cakmakci

Installation:

Die Java SDK und FX-Library sind für den Betrieb notwendig!
Die Nutzung des Chatprogramms erfordert keine weiteren importierten Librarys.
Der Import des Projekts ist von folgendem Repository-Link möglich:


Funktionen:

Im Chatprogramm ist ermöglicht:

- Ping-Test zur Überprüfung der Serveraktivität
- Konfiguration der Serverparameter -> Bemerke: Die Standardparameter sind auf . 147.86.8.31 und Port: 50001 eingestellt.->
  ......Es kann jedoch neu eingestellt werden.
- Erstellen von Nutzerprofilen
- Löschen von Nutzerprofilen
- Einloggen
- Erstellen von Chaträumen
- Löschung von Chaträumen
- Interaktion mit Nachrichten im Chatraum #
- Überprüfung von Online/Offline-Status von bestimmten Nutzern *
- Aktualisierung der Chatraumliste
- Ausloggen vom Programm


# Das versenden einer Nachricht im Chatfenster muss nicht zwingend durch das Auslösen des "senden"-Buttons erfolgen.
  Es kann auch durch die ENTER-Taste eine Nachricht verschickt werden!

* Für die Statusüberprüfung muss der Nutzername im Eingabefeld im Lobbybereich (oben rechts) eingegeben und mit ENTER-Taste bestätigt werden.
  Die Anzeige "username ist OFFLINE" kann auch im Falle eines Nicht-existenten Benutzerprofils ausgegeben werden!
